# Anix-Android

Android native app for ANIX anime streaming platform.

## Tech Stack
- Kotlin + Jetpack Compose
- Material 3 Design
- MVVM + Clean Architecture
- Hilt (DI)
- Retrofit + OkHttp
- Coil (Image Loading)
- Media3 ExoPlayer
- Lottie Animations

## Features
- Login/Register with JWT
- Browse Anime (Search, Filter by Genre, Year)
- Anime Detail with Episodes
- Watch Video (HLS/mp4 support)
- Watch History
- Favorites & Watchlist
- User Profile
- Moderator: Add Anime, Upload Video/Poster
- Admin: User Management, Stats

## API Base
- Development: `http://192.168.1.15:8081`
